# 认识元素

Python 的元组与列表类似，不同之处在于元组的元素不能修改。

元组使用小括号 **( )**，列表使用方括号 **[ ]**。

元组创建很简单，只需要在括号中添加元素，并使用逗号隔开即可。

语法格式：

```python
变量名=(元素 1, 元素 2, 元素 3...)
```

主要事项：

1. 元组的所有元素被有序地放置在一对小括号() 内，各个元素之间使用逗号，进行分隔。
2. 元组内的元素不受数据类型限制，可以是数值类型、字符串、列表、元组、字典、集合等类型。
3. 当定义一个只包含一个元素的元组时，必须在元素未尾加上逗号， 否则Python会将其视为该元素的数据类型本身，而不是一个元组。

代码示例：

```python
tuple1 = (1, 2, 3)
print(tuple1, type(tuple1))  #  (1, 2, 3) <class 'tuple'>

tuple2 = (False, 3.14, "hello", [1, 2], (1, 2, 3))
print(tuple2, type(tuple2))  # (False, 3.14, 'hello', [1, 2], (1, 2, 3)) <class 'tuple'>

nums = 1
print(nums, type(nums))  # 1 <class 'int'>
nums = (1,)
print(nums, type(nums))  # (1,) <class 'tuple'>

```

# 基本使用

## 访问元祖元素

可以通过索引（下标）来访问元祖中的元素。

代码示例：

```python
# 定义元祖
tuple = ("A", 'B', "C", "D")

# 从左往右，下标从 0 开始
print(tuple[0])
print(tuple[1])
print(tuple[2])
print(tuple[3])
print(tuple[4]) # IndexError: tuple index out of range

# 从右往左，下标从 -1 开始
print(tuple[-1])
print(tuple[-2])
print(tuple[-3])
print(tuple[-4])
print(tuple[-5]) # IndexError: tuple index out of range
```

## 元祖切片

和列表一样，元祖也支持切片操作，允许你获取元祖的一个子元祖。

代码示例：

```python
# 定义元祖
tuple = ("A", 'B', "C", "D")

# 切片操作
print(tuple[1:3]) #  ('B', 'C')
print(tuple[0:4:2]) # ('A', 'C')
print(tuple[-1:-4:-2]) # ('D', 'B')
```

## 遍历元祖

元祖是可迭代对象，可以使用 `for` 循环来遍历元祖中的每个元素。

代码示例：

```python
# 定义元祖
tuple = ("A", 'B', "C", "D")

# 遍历元祖
for i in tuple:
    print(i)
'''
A
B
C
D
''' 
```

# 元祖常见操作

- `len(tuple)`：返回元组中元素的个数
- `tuple.count(value)`：统计某个值在元组中出现的次数
- `tuple.index(value)`：返回某个值在元组中第一次出现的索引
- `value in tuple`：检查某个值是否存在于元组中
- `tuple1 + tuple2`：合并两个元组
- `tuple * n`：重复元组 `n` 次
- `max(tuple)`：返回元组中的最大值
- `min(tuple)`：返回元组中的最小值
- `tuple(iterable)`：将可迭代对象（如列表）转换为元组
- `for item in tuple:`：遍历元组中的元素
- `sorted(tuple)`：返回元组排序后的列表（元组本身不可变）
