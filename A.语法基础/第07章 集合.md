# 认识集合

集合（set）是一个无序的不重复元素序列。

集合中的元素不会重复，并且可以进行交集、并集、差集等常见的集合操作。

可以使用大括号 **{ }** 创建集合，元素之间用逗号 **,** 分隔， 或者也可以使用 **set()** 函数创建集合。

语法格式：

```python
parame = {value01,value02,...}
或者
set(value)
```

注意事项：

1. 集合的所有元素被放置在一对花括号中，各个元素之间使用逗号进行分隔
2. 在 Python 中花括号通常用于定义空字典，而空集合应该使用 `set()` 来定义

集合特点：

1. 集合是无序的，即集合中的元素没有固定的排列顺序
2. 集合中的元素必须是唯一的，即集合中不允许有重复的元素。
3. 集合中的元素必须是可哈希的，这是因为集合在内部使用哈希表来存储元素，而哈希表要求每个元素都有一个固定的哈希值以便快速查找。在Python中，大多数不可变类型(如整数、浮点数、字符串、元组等)都是可哈希的，而可变类型(如列表、字典、集合等)则是不可哈希的。

以下是一个简单实例：
```python
set1 = {1, 2, 3, 4}  # 直接使用大括号创建集合
set2 = set([4, 5, 6, 7])  # 使用 set() 函数从列表创建集合
```

> **注意：**创建一个空集合必须用 **set()** 而不是 **{ }**，因为 **{ }** 是用来创建一个空字典。

更多实例演示：

```python
>>> basket = {'apple', 'orange', 'apple', 'pear', 'orange', 'banana'}
>>> print(basket)                      # 这里演示的是去重功能
{'orange', 'banana', 'pear', 'apple'}

>>> 'orange' in basket                 # 快速判断元素是否在集合内
True

>>> 'crabgrass' in basket
False

>>> # 下面展示两个集合间的运算.
...
>>> a = set('abracadabra')
>>> b = set('alacazam')
>>> a                                  
{'a', 'r', 'b', 'c', 'd'}

>>> a - b                              # 集合a中包含而集合b中不包含的元素
{'r', 'd', 'b'}

>>> a | b                              # 集合a或b中包含的所有元素
{'a', 'c', 'r', 'd', 'b', 'm', 'z', 'l'}

>>> a & b                              # 集合a和b中都包含了的元素
{'a', 'c'}

>>> a ^ b                              # 不同时包含于a和b的元素
{'r', 'd', 'b', 'm', 'z', 'l'}
```

类似列表推导式，同样集合支持集合推导式(Set comprehension)：

```python
>>> a = {x for x in 'abracadabra' if x not in 'abc'}
>>> a
{'r', 'd'}
```

# 基本操作

## 添加元素

**语法格式如下：**

```python
s.add( x )
```

将元素 x 添加到集合 s 中，如果元素已存在，则不进行任何操作。

```
>>> thisset = set(('北京', '上海', '重庆'))
>>> thisset.update({1, 3})
>>> thisset
{1, '北京', '重庆', 3, '上海'}
>>> thisset.update([1, 4], [5, 6])
>>> thisset
{1, '北京', '重庆', 3, 4, 5, 6, '上海'}
```



# 集合常见操作

- 创建集合
  - `{}`：直接创建非空集合
  - `set()`：创建空集合或从可迭代对象转换
- 查看元素
  - `in`：检查元素是否存在
  - `not in`：检查元素是否不存在
- 修改元素
  - `add(element)`：添加单个元素
  - `update(*others)`：合并其他集合/可迭代对象（就地修改）
- 删除元素
  - `remove(element)`：删除指定元素（不存在时报错）
  - `discard(element)`：安全删除元素（不存在不报错）
  - `pop()`：随机删除并返回一个元素（集合为空时报错）
  - `clear()`：清空集合
- 集合运算
  - `union(*others)` 或 `|`：返回并集
  - `intersection(*others)` 或 `&`：返回交集
  - `difference(*others)` 或 `-`：返回差集
  - `symmetric_difference(other)` 或 `^`：返回对称差集
- 关系判断
  - `issubset(other)` 或 `<=`：判断是否为子集
  - `issuperset(other)` 或 `>=`：判断是否为超集
  - `isdisjoint(other)`：判断是否无交集
- 其他方法
  - `copy()`：返回浅拷贝
  - `len(set)`：获取元素数量
