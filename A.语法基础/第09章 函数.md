#   概述

函数是封装了实现某项功能的代码块，它们允许在程序中的任何需要时刻被调用执行，且支持重复利用。

使用函数能显著提升编程效率与代码复用性。通过提炼重复代码片段为独立函数，程序在需要时仅需简单调用，不仅减少了代码冗余，还使整体代码结构更加清晰简洁。

# 语法格式

### 定义函数

使用 `def` 关键字来定义一个函数，基本语法如下：

```python
def 函数名():
  '''函数的描述信息'''
  函数体
```

> **注意**： 函数命名应遵循标识符规定，同时，应力求名称直观反映函数功能，实现见名知意，以便于后续的编写、理解和调用。

示例：

```python
# 定义函数
def say_hello():
    print("Hello~")
    print("Nice to meet you")
```

## 调用函数

定义函数后，你可以通过函数名加上圆插号来调用它：

```python
函数名()
```

示例：

```python
# 调用函数（执行函数中的所有代码）
say_hello()
```

> **注意**：
>
> 1. 函数没有调用不会执行，并且可以反复调用，调用几次执行几次。
>
> 2. 在Python中，代码是按顺序从上到下执行的。因此，在调用一个函数之前，必须确保该函数已经被定义。如果尝试在函数定义之前调用它，Python解释器会因为找不到该函数的定义而抛出一个 NameError 异常，指出该函数名未被定义。所以，为了确保代码能够正确执行，应当遵循先定义函数后调用的原则。
>
>    ```python
>    # ❌ 错误示例
>    # 调用函数
>    say_hello()  # NameError: name 'say_hello' is not defined
>    
>    
>    # 定义函数
>    def say_hello():
>        print("Hello~")
>        print("Nice to meet you")
>    ```

# 返回值

函数执行完成后，向调用者提供的最终处理结果或数据。

语法：

```python
return 返回值
```

作用：

1. 向调用者提供最终处理结果或数据，以便进行后续的操作或逻辑判断。
2. 在函数中遇到 return 语句时，表示该函数将立即返回其后的值(或 None，如果 return 后没有跟值)，并结束当前函数的执行，不再继续执行 return 语句之后的任何代码。

返回值的三种情况：

1. 如果一个函数中没有使用 return 语句显式地返回一个值，或者 return 语句后面没有跟任何值(即只有 return 关键字)，那么该函数在结束时会隐式地返回 None 作为结果
2. 如果函数有一个返回值，那么它会在执行到 return 语句时将该值返回给调用者
3. 当函数需要返回多个值时，Python允许你以元组(tuple)的形式返回这些值。

# 参数

## 形参 & 实参

- 形参：函数定义时指定的参数。它们代表了函数期望接收的值的占位符。在函数被调用之前，形参并不实际存储任何值。形参只在函数被调用时才会被赋予具体的值，这些值来自于函数调用时提供的实参。

- 实参：调用函数时传递给函数的实际值。这些值被赋值给函数的形参，以便函数能够使用它们来执行其定义的操作。

代码示例：

```python
# 定义函数
def add(a: int, b: int):  # a,b 是形参
    return a + b

# 调用函数
print(add(1, 2))  # 1,2 是实参
```

## 函数参数

### 必备参数（位置参数）

传递参数的顺序和个数必须与函数定义时指定的参数顺序和个数一致。

代码示例：

```python
# 定义函数
def greet(name, gretting):
    print(f"{name}, {gretting}")


# 调用函数
greet()  # TypeError: greet() missing 2 required positional arguments: 'name' and 'gretting'
greet("Bob")  # TypeError: greet() missing 1 required positional argument: 'gretting'
greet("Alice", "Hi", "emmmmm")  # TypeError: greet() takes 2 positional arguments but 3 were given
```

### 缺省参数（默认参数）

在函数定义时，可以为参数指定默认值。如果在函数调用时没有提供该参数的值，则使用默认值。

代码示例：

```python
# 定义函数
def greet(name, gretting="Hello"):
    print(f"{name}, {gretting}")

# 调用函数
greet("Bob")  # Bob, Hello
greet("Bob", "Hi")  # Bob, Hi
```

> 注意：在Python中，定义函数时，非默认参数必须放在默认参数之前。

错误示例：

```python
# 错误的函数定义
def greet(gretting="Hello", name):
    print(f"{name}, {gretting}")
# SyntaxError: parameter without a default follows parameter with a default
```

### 关键字参数

使用 `**kwargs` 可以允许函数接收任意数量的关键字参数。这些参数会被收集到一个名为 kwargs 的字典中(同样你可以使用任何变量名，但 kwargs 是惯例)。

```python
# 定义函数
def greet(**kwargs):
    print(kwargs)

# 调用函数
greet()  # {}
greet(name="Alice", age=30)  # {'name': 'Alice', 'age': 30}
```

关键字参数的作用在于显著地增强函数的灵活性和可扩展性，特别适用于那些需要处理多种输入参数，其中部分参数为必填而另一部分为可选的场景。以用户注册功能为例，通常会有用户名和年龄等必填项，以确保用户的基本信息完整，然而，不同场景下可能还需要收集用户的邮箱、电话号码、性别、地址等额外信息，这些信息并非每次注册都必需，因此适合作为可选参数。

示例：

```python
# 定义函数
def register_user(name, age, **kwargs):
    # 用户名和年龄是必填项
    print(f"注册用户:{name}, 年龄:{age}")
    # 处理可选的关键字参数
    for key, value in kwargs.items():
        print(f"额外信息：{key} = {value}")

# 调用函数，只提供必填项
register_user("Alice", 20)
# 调用函数，提供必填项和可选项
register_user("Bob", 25, email="bob@example.com", phone="123-456-7890")

'''
注册用户:Alice, 年龄:20
注册用户:Bob, 年龄:25
额外信息：email = bob@example.com
额外信息：phone = 123-456-7890
'''
```

# 变量作用域

指程序代码能够访问该变量的区域，也就是变量生效的范围。

## 分类

根据变量作用域的不同，我们可以将变量主要分为两大类：**局部变量** 和 **全局变量**

### 局部变量

在Python中，局部变量是在函数内部定义的变量。这些变量仅在定义它们的函数内部可见和可用。一旦退出该函数，它们就超出了作用域，变得不可访问。

示例：

```python
# 定义函数
def func():
    x = "局部变量"
    print(x)


func()  # 局部变量
print(x)  # NameError: name 'x' is not defined
```

### 全局变量

在Python中，全局变量是在函数外部定义的变量。这些变量在程序的整个执行过程中都是可访问的。

```python
x = "全局变量"

def f():
    print(f"func：{x}")

f()  # func：全局变量
print(f"外部：{x}")  # 外部：全局变量
```

> **注意**：在 Python中，如果全局变量和局部变量命名相同，那么局部变量会覆盖同名的全局变量，在局部作用域内，只有局部变量的值会被访问到。但是，在全局作用域或其他不包含该局部变量的作用域中，全局变量的值仍然保持不变。 

##  `global` vs. `nonlocal`

### `global`

在 Python 中，`global` 关键字用于在函数内部声明全局变量。 

语法格式：

```python
global 变量名
```

代码示例：

```python
# 定义全局变量
count = 0

def increment():
    # 声明 count 是全局变量
    global count
    # 修改全局变量的值
    count += 1

increment()
print(count)  # 1
```

```python
def func():
    # 声明 count 是全局变量，可以在外部访问
    global count
    count = 10

func()
print(count) # 10
```

思考：如果需要声明多个全局变量怎么办？

如果需要声明多个全局变量，在Python中，只需在函数内部为每个全局变量使用 `global` 关键字即可。 `global` 关键字后面可以跟随一个或多个由逗县分隔的变量名。

```python
# 定义函数
def modify_globals():
    global var1, var2, var3
    var1 = 10
    var2 = "Hello"
    var3 = [1, 2, 3]

# 在调用函数之前，这些变量在全局作用域中不存在
print(var1)  # NameError: name 'var1' is not defined. Did you mean: 'vars'?
# 调用函数
modify_globals()
# 现在，这些变量已经在全局作用域中被创建并赋值
print(f"var1：{var1}")
print(f"var2：{var2}")
print(f"var3：{var3}")
```

### `nonlocal`

`nonlocal` 关键字用于在内层函数中引用并可能修改外层函数作用域中的局部变量，同时确保这个变量不是在当前函数内部新创建的局部变量，也不是全局变量。

代码示例：

```python
# 定义函数
def outer():                # 外层函数
    a = 10                  # 外层函数的局部变量
    print(f"outer1:{a}")    # outer1:10
    def inner():            # 内层函数
        nonlocal a          # 声明 a 是外层函数的局部变量
        a = 20              # 修改外层函数的局部变量的值
        print(f"inner:{a}") # inner:20

    inner()  							  # 调用内层函数，修改并输出 a 的值
    print(f"outer2:{a}")    # outer2:20
# 调用函数
outer()                     # 调用外层函数
```
