# 认识列表

在 Python 中，列表是一种非常强大且灵活的数据结构，用于存储一系列有序的元素。

语法格式：

```python
变量名=[元素 1, 元素 2, 元素 3, ...]
```

注意：

1. 列表的所有元素被有序地放置在一对 **中括号** 内，各个元素之间使用 **逗号** `,` 进行分隔。

   ```python
   # 定义列表
   li = [1, 2, 3]
   
   # 输出列表及其类型
   print(li, type(li)) # [1, 2, 3] <class 'list'>

2. 列表内的元素不受数据类型限制，可以是数值类型、字符串、列表、元组、字典、集合等 类型。

   ```python
   # 定义列表
   li = [False, 3.5, 'Python', [1, 2]]
   
   # 输出列表及其类型
   print(li, type(li)) # [False, 3.5, 'Python', [1, 2]] <class 'list'>
   ```

# 基本使用

## 访问列表中的值

与字符串的索引一样，列表索引从 **0** 开始，第二个索引是 **1**，依此类推。

通过索引列表可以进行截取、组合等操作。

```python
list = ["red", "green", "blue", "yellow", "white", "black"]
print(list[0])
print(list[1])
print(list[2])
```

索引也可以从尾部开始，最后一个元素的索引为 **-1**，往前一位为 **-2**，以此类推。

```python
list = ["red", "green", "blue", "yellow", "white", "black"]
print(list[-1])
print(list[-2])
print(list[-3])
```

使用下标索引来访问列表中的值，同样你也可以使用方括号 **[]** 的形式截取字符，如下所示：

```python
nums = [10, 20, 30, 40, 50, 60, 70, 80, 90]
print(nums[0:4])
```

以上实例输出结果：

```
[10, 20, 30, 40]
```

使用负数索引值截取：

```python
list = ["成都", "重庆", "贵阳", "武汉", "桂林"]

# 读取第二位
print("list[1]: ", list[1])
# 从第二位开始（包含）截取到倒数第二位（不包含）
print("list[1:-2]: ", list[1:-2])
```

以上实例输出结果：

```
list[1]:  重庆
list[1:-2]:  ['重庆', '贵阳']
```

## 遍历列表

列表是可迭代对象，可以使用 `for` 循环来遍历列表中的每个元素

```python
# 定义列表
li = ['A', 'B', 'C', 'D']

for i in li:
  print(i)


'''
A
B
C
D
'''
```

# 常见操作

## 添加元素

- [`list.append(obj)`](https://www.runoob.com/python3/python3-att-list-append.html)：在列表末尾添加新的对象
- [`list.extend(seq)`](https://www.runoob.com/python3/python3-att-list-extend.html)：在列表末尾一次性追加另一个序列中的多个值（用新列表扩展原来的列表）
- [`list.insert(index, obj)`](https://www.runoob.com/python3/python3-att-list-insert.html)：将对象插入列表

> **注意**：如果仅仅是在列表的未尾添加元素，通常推荐使用 `append` 方法，效率更高。因为它不需要计算索引位置，也不需要移动列表中的其他元素。

## 更新元素

通过指定元素的索引(下标)来修改列表中的元素。

语法格式：

```python
list[index] = value
```

参数说明：

- `index`：要修改的元素的索引位置
- `value`：要赋值给指定索引位置的新值，可以是任意数据类型。

## 查找元素

- `in`：检查一个元素是否存在于列表中，如果元素存在，返回 `True`，否则返回 `False`。
- `not in`：检查一个元素是否不存在于列表中，如果元素不存在，返回 `True`，否则返回 `False`。
- [`list.index(obj)`](https://www.runoob.com/python3/python3-att-list-index.html)：从列表中找出某个值第一个匹配项的索引位置
- [`list.count(obj)`](https://www.runoob.com/python3/python3-att-list-count.html)：统计某个元素在列表中出现的次数

## 删除元素

- `del`：删除列表中的元素
- [`list.remove(obj)`](https://www.runoob.com/python3/python3-att-list-remove.html)：移除列表中某个值的第一个匹配项
- [`list.clear()`](https://www.runoob.com/python3/python3-att-list-clear.html)：清空列表
- [`list.pop([index=-1])`](https://www.runoob.com/python3/python3-att-list-pop.html)：移除列表中的一个元素（默认最后一个元素），并且返回该元素的值

## 复制列表

- [`list.copy()`](https://www.runoob.com/python3/python3-att-list-copy.html)

## 排序

- [`list.sort( key=None, reverse=False)`](https://www.runoob.com/python3/python3-att-list-sort.html)：对原列表进行排序
- [`list.reverse()`](https://www.runoob.com/python3/python3-att-list-reverse.html)：反向列表中元素

# 列表推导式

列表推导式是Python中一种简洁创建列表的方法。它可以用一行代码完成原本需要多行循环才能实现的功能，让代码更简洁易读。

## 基本语法

列表推导式的基本语法如下：

```python
[表达式 for 元素 in 可迭代对象]
```

简单示例：创建1-10的数字列表

```python
# 传统方法
numbers = []
for i in range(1, 11):
    numbers.append(i)

# 列表推导式
numbers = [i for i in range(1, 11)]
```

## 进阶用法

### 元素转换

需求：将列表中每个元素乘以5

```python
nums = [11, 22, 33, 44, 55, 66]

# 传统方法
result = []
for num in nums:
    result.append(num * 5)

# 列表推导式
result = [num * 5 for num in nums]
```

### 条件过滤

可以在列表推导式中加入条件表达式，以过滤出满足条件的元素

```python
[expression for item in iterable if confition] 
```

需求：筛选出列表中的偶数

```python
nums = [11, 22, 33, 44, 55, 66]

# 传统方法
evens = []
for num in nums:
    if num % 2 == 0:
        evens.append(num)

# 列表推导式
evens = [num for num in nums if num % 2 == 0]
```

## 语法总结

| 语法                                 | 说明     | 示例                               |
| :----------------------------------- | :------- | ---------------------------------- |
| `[x for x in iterable]`              | 基本形式 | `[i for i in range(5)]`            |
| `[x*2 for x in iterable]`            | 元素转换 | `[n*2 for n in [1,2,3]]`           |
| `[x for x in iterable if condition]` | 条件过滤 | `[n for n in range(10) if n%2==0]` |
