# 输入与输出

## 输入

`print` 函数

```python
print(1, 2, 3, sep="*", end="!")
```

参数解读：

- `1`，`2`，`3`：输出列表
- `sep`：分隔符，默认空格
- `end`：结束符

## 输出

在 Python中，程序获取用户输入数据的功能通常被称为输入操作，这一功能是通过内置函数 `input` 来实现。

语法如下：

```python
input(__prompt=None)
```

参数解读：

- `__prompt`：提示信息，会在控制台显示，可省略

代码示例：

```python
name = input('请输入姓名：') # 将用户输入的姓名信息保存到变量 name 中
print(f'你好呀，{name}!')
```

> **注意**：在 Python中，`input` 函数默认将用户的所有输入视为字符串类型，无论输入内容的实际性质如何。

# 标识符

在编程语言中，标识符(ldentifier)是一个用于识别变量、函数、类、模块或其他对象名称的字符串。在 Python 中，标识符用于给变量、函数、类和其他对象命名。

在Python中，标识符的使用受到严格语法规则的约束，违反这些规则将直接导致语法错误，影响代码的执行。关于Python标识符的规定，主要遵循以下原则:

📖 **强制规则（必须遵守）**

1. **字符范围**：仅含字母（`A-Z/a-z`）、数字（`0-9`）、下划线（`_`），**不能以数字开头**。
2. **禁止关键字**：如 `if`、`for`、`class` 等（可通过 `import keyword; print(keyword.kwlist)` 查看全部）。
3. **区分大小写**：`name` 和 `Name` 是不同变量。

📖 **命名规范（PEP 8 推荐）**

1. **变量/函数名**：小写 + 下划线（蛇形命名法），如 `user_age`。
2. **类名**：首字母大写驼峰法（`CamelCase`），如 `UserProfile`。
3. **常量名**：全大写 + 下划线，如 `MAX_LIMIT`。

📖 **最佳实践**

见名知意

> **总结**：变量名需合法（字母/下划线开头，非关键字），推荐小写下划线（`snake_case`），类名用驼峰（`CamelCase`），常量全大写（`UPPER_CASE`）。

# 数字

## 整型 `int`

整型用于存储任意大小的整数

```python
# 定义整型变量
num = 100
print(num, type(num)) # type(x) 获取 x 的数据类型

# 整型可以表示非常大/小的数
num2 = 10000000000000000000000000000
print(num2, type(num2))
```

## 浮点型 `float`

浮点型用于存储带有小数部分的数值。在Python中，当你定义一个包含小数点的数字时，它会自动被视为浮点型( `float` )。对于非常大或非常小的数，只要它们包含小数点或者使用了科学计数法(例如 `e` 或 `E` 表示10的幂)，它们就会被视为浮点数。

```python
# 定义浮点型变量
pi = 3.14
print(pi, type(pi))

# 浮点数可以非常大或非常小
num = 1234567890123456789012345.0
print(num)
```

## 布尔型 `bool`

布尔类型（`bool`）包含两个明确且固定的值:`True` (真)和 `False`(假)，它常用于逻辑判断、条件控制等编程场景中，以简化代码逻辑和增强可读性。

```python
# 定义布尔型变量
v1 = True
print(v1, type(v1))

v2 = False
print(v2, type(v2))
```

注意事项：

1. 严格区分大小写，即 `True !== true`，`False !== false` 
2. 布尔值可以被隐式转换，用于数据运算，`True` — `1`，`False` — `0`

## 复数型 `complex`

Python 中的 `complex` 类型用于表示复数。复数是一个基础性的数学概念，由实部和虚部组成，虚部后面紧跟一个 `j`（或 `J`，大小写均可）作为后缀。广泛应用于数据计算领域，利用其独特性质解决仅用实数难以处理的复杂问题,

```python
# 定义复数型变量
c = 3 + 4j  # j(或J) 在复数中作为固定的虚数单位，不可更改
print(c, type(c))
```

# 格式化输出

在 Python 中，格式化输出通过将值插入字符串占位符来生成格式化的字符串，让数据展示更加清晰、易读，并符合特定格式。

```python
name = '张三'
age = 18
sex = "女"


# 需求：要求按照如下格式输出内容
# 我的姓名：张三，我的年龄：18，我的性别：女

# 1. 原始方式：借助 print 函数中的 *arg 参数
print('我的姓名：%s，我的年龄：%d，我的性别：%s' % (name, age, sex))
print('我的姓名：', name, '，我的年龄：', age, '，我的性别：', sex, sep='') 


# 2. 使用格式化输出：str.format()
print('我的姓名：{}，我的年龄：{}，我的性别：{}'.format(name, age, sex))

# 3. 使用格式化输出：f-string（推荐）

# -- 基础用法
print(f'我的姓名：{name}，我的年龄：{age}，我的性别：{sex}')  
print(F'我的姓名：{name}，我的年龄：{age}，我的性别：{sex}')  


# -- 支持表达式
n1 = 3
n2 = 4

# 输出：3 * 4 = 12
print(f'{n1} * {n2} = {n1 * n2}')


# -- 支持整数补位
sid = 1
print(f'我的学号：{sid:3d}')  # 设置 3 位数，不足默认前面补空格
print(f'我的学号：{sid:03d}') # 设置 3 位数，不足前面补 0
print(f'我的学号：123')

pi = 3.1415926
print(f'圆周率：{pi}')
print(f'圆周率：{pi:.2f}')  # 保留 2 位小数，超出 2 位小数则遵循四舍五入的原则，不足两位小数则后面补 0

num = 1.5
print(f'num = {num:.3f}') # 保留 3 位小数，超出 3 位小数则遵循四舍五入的原则，不足两位小数则后面补 0
```

# 运算符

## 算术运算符

| #    | 不爱三个         | 备注                                                         |
| ---- | ---------------- | ------------------------------------------------------------ |
| `+`  | 加法运算         | `a + b`                                                      |
| `-`  | 减法运算         | `a - b`                                                      |
| `*`  | 乘法运算         | `a * b`                                                      |
| `/`  | 除法运算         | `a / b`，返回浮点数结果，哪怕能被整除                        |
| `//` | 整除运算         | `a // b`，将第一个数除以第二个数，并返回商的整数部分，向下取整（即向左取整） |
| `%`  | 模运算（取余数） | `a % b`                                                      |
| `**` | 指数运算         | `a ** b`，类似于 js 中的 `pow(a, b)`，即 `a` 的 `b` 次方     |

优先级：

1. 先乘除后加减
2. 同级运算符从左往右计算
3. 可以用 `()` 调整计算的优先级

运算符优先级排序：`**` > `*`，`/`，`//`，`%` > `+`，`-`

> 注意：任何数和一个浮点数运算，得到的结果都是浮点数。

## 赋值运算符

在 Python中，赋值运算符主要用于给变量赋值。在使用时，将右侧的值赋给左侧的变量，右侧也可以在进行某些运算后再赋值给左侧的变量。

| 运算符 | 描述             | 实例                                    |
| ------ | ---------------- | --------------------------------------- |
| `=`    | 简单的赋值运算符 | `c = a + b` 将 `a + b` 的结果赋值为 `c` |
| `+=`   | 加法赋值运算符   | `c += a` 等效于 `c = c + a`             |
| `-=`   | 减法赋值运算符   | `c -= a` 等效于 `c = c - a`             |
| `*=`   | 乘法赋值运算符   | `c *= a` 等效于 `c = c * a`             |
| `/=`   | 除法赋值运算符   | `c /= a` 等效于 `c = c / a`             |
| `%=`   | 取模赋值运算符   | `c %= a` 等效于 `c = c % a`             |
| `**=`  | 幂赋值运算符     | `c **= a` 等效于 `c = c ** a`           |
| `//=`  | 取整除赋值运算符 | `c //= a` 等效于 `c = c // a`           |

Python 编程中，在适用且能提升代码清晰性的情况下，应优先考虑使用复合赋值运算符(如 `+=`)，因为它们能够使代码更加简洁和易读。

> **注意**：复合赋值运算符只能用于已经存在的变量。这是因为它们的工作方式是首先访问变量的当前值，然后执行指定的运算，并将结果重新赋值给该变量。

## 比较运算符

Python 中的比较运算符用于比较两个值或表达式之间的关系，并根据比较结果返回布尔值 `True` 或 `False`，这些运算符在条件语句(如`if` 语句)、循环控制以及数据筛选等场景中非常有用。

| 运算符 | 示例     | 说明                                                         |
| ------ | -------- | ------------------------------------------------------------ |
| `==`   | `a == b` | 如果两边的值相等，则返回 `True`，否则返回 `False`            |
| `!=`   | `a != b` | 如果两边的值不相等，则返回 `True`，否则返回 `False`          |
| `>`    | `a > b`  | 如果左边的值大于右边的值，则返回 `True`，否则返回 `False`    |
| `>=`   | `a >= b` | 如果左边的值大于或等于右边的值，则返回 `True`，否则返回 `False` |
| `<`    | `a < b`  | 如果左边的值小于右边的值，则返回 `True`，否则返回 `False`    |
| `<=`   | `a <= b` | 如果左边的值小于或等于右边的值，则返回 `True`，否则返回 `False` |

## 逻辑运算符

通过使用逻辑运算符，开发者可以将多个条件表达式组合起来，形成复杂的条件判断语句。这些语句能够精确地描述程序在不同条件下的行为，确保在满足特定条件时执行相应的代码块，而在不满足条件时则跳过这些代码块或执行其他备选路径。

| 运算符        | 示例      | 说明                                                         |
| ------------- | --------- | ------------------------------------------------------------ |
| 逻辑与：`and` | `a and b` | 当所有条件都为 `True` 时，整个表达式结果才为 `True`，否则为 `False` |
| 逻辑或：`or`  | `a or b`  | 只要其中一个条件为 `True`，整个表达式结果就为 `True`，否则为 `False` |
| 逻辑非：`not` | `not a`   | 反转条件结果：若条件为 `True` 则结果为 `False`，反之则为 `True` |

代码示例：

```python
x = True
y = False
print(x and y)  # 输出: False
print(x or y)   # 输出: True
print(not x)    # 输出: False
```

## 成员运算符

除了以上的一些运算符之外，Python还支持成员运算符，测试实例中包含了一系列的成员，包括字符串，列表或元组。

| 运算符 | 描述                                                    | 实例                                              |
| :----- | :------------------------------------------------------ | :------------------------------------------------ |
| in     | 如果在指定的序列中找到值返回 True，否则返回 False。     | x 在 y 序列中 , 如果 x 在 y 序列中返回 True。     |
| not in | 如果在指定的序列中没有找到值返回 True，否则返回 False。 | x 不在 y 序列中 , 如果 x 不在 y 序列中返回 True。 |

代码示例：

```python
a = 10
b = 20
list = [1, 2, 3, 4, 5]

if a in list:
    print("1 - 变量 a 在给定的列表中 list 中")
else:
    print("1 - 变量 a 不在给定的列表中 list 中")

if b not in list:
    print("2 - 变量 b 不在给定的列表中 list 中")
else:
    print("2 - 变量 b 在给定的列表中 list 中")

# 修改变量 a 的值
a = 2
if a in list:
    print("3 - 变量 a 在给定的列表中 list 中")
else:
    print("3 - 变量 a 不在给定的列表中 list 中")
```

输出结果：

```
1 - 变量 a 不在给定的列表中 list 中
2 - 变量 b 不在给定的列表中 list 中
3 - 变量 a 在给定的列表中 list 中
```

# 转义字符

在 Python中，转义字符用于在字符串中表示那些通常具有特殊含义的字符。

| 转义字符 | 说明                                                   | 等效按键/场景示例         |
| -------- | ------------------------------------------------------ | ------------------------- |
| `\n`     | 换行符，将光标位置移到下一行开头                       | Enter 键                  |
| `\r`     | 回车符，将光标位置移到本行开头                         | 旧系统换行符（如Windows） |
| `\t`     | 水平制表符（Tab键），一般相当于4个空格，常用语对齐文本 | Tab 键                    |
| `\b`     | 退格，将光标位置移到前一列                             | Backspace 键              |
| `\\`     | 反斜线（用于转义字符本身）                             | 路径字符串（如 `C:\\`）   |
| `\'`     | 单引号（在单引号字符串内使用）                         | `'It\'s Python'`          |
| `\"`     | 双引号（在双引号字符串内使用）                         | `"He said \"Hi\""`        |
| `\`      | 续行符，将一行代码分成多行书写（需在行尾）             | 长字符串或表达式换行      |

代码示例：

```python
# \t - 水平制表符
print('w\t*')
print('ww\t*')
print('www\t*')
print('wwww\t*')
print('wwwww\t*')
print('wwwwww\t*')
print('wwwwwww\t*')
print('wwwwwwww\t*')
print('wwwwwwwww\t*')
print('wwwwwwwwww\t*')


# \n - 换行
print('这是第一行\n这是第二行\n这是第三行')

# \' - 单引号
print('He said, "What\'s there?"')

# \" - 双引号
print("He said, \"What's there?\"")

# r"" - 原始字符串
print(r"C:\Users\example\Documents")
```
