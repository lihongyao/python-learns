# 创建数据库

PostgreSQL 创建数据库可以用以下三种方式：

1、使用 **CREATE DATABASE** SQL 语句来创建。

```postgresql
CREATE DATABASE <dbname>  ...
```

2、使用 **createdb** 命令来创建。

```shell
$ createdb <dbname>
```

3、使用 **pgAdmin/DBeaver** 工具。

# 查看 / 切换数据库

（1）查看已经存在的数据库

```postgresql
postgres-# \l
```

```shell
$ psql -l
```

（2）查看当前链接数据库

```postgresql
postgres=# \conninfo
postgres=# SELECT current_database(); 
```

（3）使用 **`\c + 数据库名`** 来进入数据库：

```postgresql
postgres-# \c mydatabase
psql (17.0, server 17.4 (Homebrew))
You are now connected to database "mydatabase" as user "root".
```

# 删除数据库

PostgreSQL 删除数据库可以用以下三种方式：

1、使用 **DROP DATABASE** SQL 语句来删除。

```postgresql
DROP DATABASE [ IF EXISTS ] <dbname>
```

2、使用 **dropdb** 命令来删除。

```shell
$ dropdb <dbname>
```

3、使用 **pgAdmin/DBeaver** 工具。

# 链接数据库

```shell
$ psql # 链接到同名数据库
$ psql -U <username> -d <dbname>
$ psql -h <host> -p <port> -U  <username> <dbname>
```
