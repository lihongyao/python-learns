# 概述

PostgreSQL 是一款功能强大的 **开源对象-关系型数据库管理系统（ORDBMS）**，起源于1986年加州大学伯克利分校的POSTGRES项目，经过30多年发展已成为最先进的开源关系型数据库之一，以下是其核心特点：

**核心特性**

- **标准兼容**：支持SQL规范（包括窗口函数、CTE等），提供ACID事务、多版本并发控制（MVCC）
- 丰富数据类型：除基础类型外，支持JSON、数组、几何图形、IP地址等复杂类型，以及自定义类型扩展
- **高性能**：列存储、并行查询、JIT编译等技术优化分析型查询速度
- **可扩展性**：通过插件支持全文检索（Tsearch2）、地理空间数据（PostGIS）、时序数据等场景

**技术优势**

- **多版本并发控制（MVCC）**：高并发下读写互不阻塞，适合金融、电商等高并发场景
- **安全机制**：支持SSL加密、行级安全策略、角色权限管理等
- **跨平台**：兼容Linux、Windows、macOS等主流操作系统

**应用场景**

- **数据分析**：支持复杂查询和实时分析，常与Greenplum等数据仓库集成
- **Web服务**：作为Django、FastAPI等框架的后端数据库
- **地理信息系统（GIS）**：通过PostGIS扩展处理空间数据
- **金融与物联网**：严格的事务一致性和实时数据处理能力

**开源生态**

- **许可证**：采用自由的BSD许可证，允许商用和二次开发
- **社区支持**：全球活跃开发者社区持续贡献新功能，如最新版本已支持AI向量检索

PostgreSQL以其**功能全面、稳定可靠**的特性，成为企业级应用和新兴技术（如AI、区块链）的首选数据库之一

# ORDBMS 术语

在我们开始学习 PostgreSQL 数据库前，让我们先了解下 ORDBMS 的一些术语：

- **数据库:** 数据库是一些关联表的集合。
- **数据表:** 表是数据的矩阵。在一个数据库中的表看起来像一个简单的电子表格。
- **列:** 一列(数据元素) 包含了相同的数据, 例如邮政编码的数据。
- **行：**一行（=元组，或记录）是一组相关的数据，例如一条用户订阅的数据。
- **冗余**：存储两倍数据，冗余降低了性能，但提高了数据的安全性。
- **主键**：主键是唯一的。一个数据表中只能包含一个主键。你可以使用主键来查询数据。
- **外键：**外键用于关联两个表。
- **复合键**：复合键（组合键）将多个列作为一个索引键，一般用于复合索引。
- **索引：**使用索引可快速访问数据库表中的特定信息。索引是对数据库表中一列或多列的值进行排序的一种结构。类似于书籍的目录。
- **参照完整性:** 参照的完整性要求关系中不允许引用不存在的实体。与实体完整性是关系模型必须满足的完整性约束条件，目的是保证数据的一致性。

# 环境配置

> 提示：本手册以 macOS 系统为例

1. 安装

   ```shell
   $ brew install postgresql@17
   ```

2. 配置环境变量

   由于 PostgreSQL 17 是 `keg-only` 安装，需手动添加路径到环境变量：

   ```python
   # 将以下命令粘贴到终端执行（永久生效）
   echo 'export PATH="/opt/homebrew/opt/postgresql@17/bin:$PATH"' >> ~/.zshrc
   source ~psql -U root -d postgres/.zshrc
   ```

3. 启动 PostgreSQL 服务，选择以下任一方式启动：

   ```shell
   # 方式一：后台服务（推荐）
   brew services start postgresql@17  # 开机自启
   # 方式二：手动启动
   LC_ALL="C" /opt/homebrew/opt/postgresql@17/bin/postgres -D /opt/homebrew/var/postgresql@17
   ```

4. 验证安装

   ```shell
   $ psql --version
   psql (PostgreSQL) 17.4 (Homebrew)
   ```

5.  创建用户和数据库（可选）

   ```shell
   $ createuser --interactive  # 按提示创建用户（如设为超级用户）
   $ createdb                  # 创建同名数据库，后续在终端直接输入 psql 将自动连接到同名数据库中
   $ psql                      # 链接到同名数据库中
   ```

6. 链接数据库

   ```shell
   $ psql -U root -d postgres   #  以 root 身份链接postgres数据库
   ```

> **提示**：退出命令行模式使用快捷键：<kbd>Ctr</kbd> + <kbd>D</kbd>

# 基本操作

```shell
postgres=# \conninfo         -- 显示当前连接详情（用户、数据库、主机等）
postgres=# \h 指令            -- 帮助指南
postgres=# \q                -- 退出    
postgres=# SELECT version(); -- 检查版本信息
```

# SQL语句

一个 SQL 语句通常包含了关键字、标识符（字段）、常量、特殊符号等，下面是一个简单的 SQL 语句：

```postgresql
SELECT id, name FROM users
```

|          | SELECT |    id, name     |           FROM           | users  |
| :------: | :----: | :-------------: | :----------------------: | ------ |
| 符号类型 | 关键字 | 标识符（字段）  |          关键字          | 标识符 |
|   描述   |  命令  | id 和 name 字段 | 语句，用于设置条件规则等 | 表名   |

> **提示**：PostgreSQL 命令参考 [这里  ↪](https://www.runoob.com/postgresql/postgresql-syntax.html)
