# 概述

## 含义

正则表达式（Regular Expression）又称 **规则表达式**，是一种文本模式（pattern）。它定义了一系列规则来匹配字符串中的特定模式。这些模式可以非常灵活和复杂，能够用于识别和处理各种文本数据。

它是强大的字符串处理工具。在软件开发中，我们经常需要查找、验证或提取符合特定规则的字符串。例如，在网页爬虫或数据抓取任务中，我们可能需要从大量文本中提取出所有有效的邮箱地址或电话号码。这时，正则表达式就派上了大用场，它允许我们以一种高效目准确的方式来完成这些任务。

Python自1.5版本开始内置了`re` 模块，该模块提供了完整的正则表达式功能。要使用正则表达式，首先需要导入Python 的 `re` 模块。通过 `re` 模块提供的一系列函数，我们可以轻松地在 Python 程序中进行正则表达式的匹配、搜索、替换等操作。

示例：

```python
import re
```

## 特点

1. **语法复杂性**：正则表达式的语法相对复杂，包含了许多特殊字符和元字符，这些字符具有特定的含义和用途。由于这种复杂性，正则表达式的可读性通常较差，尤其是对于那些不熟悉其语法的开发者来说。然而，正是这种复杂性赋予了正则表达式强大的匹配能力，能够处理各种复杂的文本模式。
2. **通用性**：正则表达式的强大之处在于其通用性。几平所有的现代编程语言都支持正则表达式，包括但不限干Python、javaScript、java、C#等。这意味着开发者可以在不同的编程环境中使用相似的语法和规则来编写正则表达式，实现跨语言的文本处理功能。这种通用性使得正则表达式成为处理字符串数据时不可或缺的工具之一。

## 简单使用

步骤如下：

1. 导入模块

2. 使用 `match（）` 进行匹配操作

   语法格式：

   ```python
   re.match(pattern, string, flags=0) 
   ```

   参数说明：

   - `pattern`：要匹配的正则表达式模式
   - `string`：要搜索的字符串
   - `flags`（可选）：标志位，用于控制正则表达式的匹配方法，如：是否区分大小写，是否多行匹配等，如果省略，则默认为 0 ，表示不使用任何标志。

   示例：

   ```python
   import re
   
   res = re.match(r"o", "hello")
   print(res)  # None
   
   
   res = re.match(r"h", "hello")
   print(res)  # <re.Match object; span=(0, 1), match='h'> (匹配成功)
   ```

3. 匹配成功，通过 `group` 方法提取数据

   **注意**：当使用 `group` 方法时，如果匹配失败（即 `re.match()` 返回 `None`），尝试调用 `group` 方法将引发 `AttributeError`。因此，在调用 `group` 方法之前，应该先检查匹配对象是否为 None。

   ```python
   import re
   
   res = re.match(r"h", "hello")
   print(res)  # <re.Match object; span=(0, 1), match='h'> (匹配成功)
   print(res.group())  # h
   
   
   res = re.match(r"o", "hello")
   print(res)  # None (匹配失败)
   print(res.group())  # AttributeError: 'NoneType' object has no attribute 'group'
   
   
   # 最佳实践
   res = re.match(r"o", "hello")
   if res:
       print(res.group())
   else:
       print("匹配失败")
   ```

# 匹配函数

Python的 `re` 模块提供了多种正则表达式匹配函数，以下是常用的函数及其功能：

## 基础匹配函数

1. **`re.match(pattern, string)`**

   - **功能**：从字符串**开头**匹配正则表达式，成功返回 `Match` 对象，否则返回 `None`。

   - **示例**：

     ```python
     import re
     result = re.match(r'\d+', '123abc')
     print(result.group()) if result else print("无匹配")  # 输出: 123
     ```

2. **`re.search(pattern, string)`**

   - **功能**：扫描整个字符串，返回 **第一个** 匹配的 `Match` 对象，否则返回 `None`。

   - **示例**：

     ```python
     result = re.search(r'\d+', 'abc123def')
     print(result.group()) if result else print("无匹配")  # 输出: 123
     ```

3. **`re.fullmatch(pattern, string)`**

   - **功能**：要求**整个字符串完全匹配**正则表达式，否则返回`None`。

   - **示例**：

     ```python
     result = re.fullmatch(r'\d+', '123')
     print(result.group()) if result else print("无匹配")  # 输出: 123
     ```

## 查找与迭代函数

1. **`re.findall(pattern, string)`**

   - **功能**：返回所有**非重叠匹配**的子串列表。

   - **示例**：

     ```python
     print(re.findall(r'\d+', '123abc456def'))  # 输出: ['123', '456']
     ```

2. **`re.finditer(pattern, string)`**

   - **功能**：返回所有匹配的**迭代器**（生成`Match`对象）。

   - **示例**：

     ```python
     for match in re.finditer(r'\d+', '123abc456def'):
         print(match.group())  # 输出: 123, 456
     ```

## 替换与分割函数

1. **`re.sub(pattern, repl, string)`**

   - **功能**：替换所有匹配的子串为`repl`。

   - **示例**：

     ```python
     print(re.sub(r'\d+', 'NUM', '123abc456def'))  # 输出: NUMabcNUMdef
     ```

2. **`re.split(pattern, string)`**

   - **功能**：按正则表达式分割字符串，返回列表。

   - **示例**：

     ```python
     print(re.split(r'\d+', '123abc456def'))  # 输出: ['', 'abc', 'def']
     ```

## 高级功能

1. **`re.compile(pattern)`**

   - **功能**：预编译正则表达式，提升重复使用效率。

   - **示例**：

     ```python
     pattern = re.compile(r'\d+')
     print(pattern.findall('123abc456def'))  # 输出: ['123', '456']
     ```

## 扩展

1. 关键区别：
   - **`match` vs `search`**：`match` 仅检查开头，`search` 扫描整个字符串
   - **`findall` vs `finditer`**：前者返回列表，后者返回迭代器
   - **`sub` vs `split`**：前者替换匹配项，后者按匹配项分割
2. 适用场景
   - **验证格式**：`match`或`fullmatch`（如检查密码规则）。
   - **提取数据**：`findall`或`finditer`（如爬虫抓取文本）。
   - **文本处理**：`sub`或`split`（如清洗日志文件）。

# 基础语法

## 转义字符

使用正则表达式去检索某些特殊字符的时候，需要用到转义字符，否则检索不到结果，甚至会报错，在 Python 中，`\` 具有转义的意思，会对紧随其后的字符进行转义，如果我们想使用普通的 `\` ，需要再使用一个 `\` 对它进行转义。

代码示例：

```python
import re

content = "abc$def(123(456))"
regex = r"\(456"

result = re.findall(regex, content)
print(result)  # ['(456']

result = re.match(regex, content)
print(result)  # None
```

> **提示**：需要用到转义符号的常见字符如下：. * + ( ) $ / \ ? [ ] ^

## 字符匹配符

### 字符匹配符（正则表达式）

| 符号   | 名称           | 实例     | 说明                                                         |
| ------ | -------------- | -------- | ------------------------------------------------------------ |
| `[ ]`  | 字符集合       | `[abc]`  | 匹配方括号内**任意一个字符**（如 `a`、`b` 或 `c`）           |
| `[^ ]` | 排除型字符集合 | `[^abc]` | 匹配**不在**方括号内的任意字符（如数字、特殊符号或其他字母） |
| `-`    | 范围连字符     | `[a-z]`  | 匹配指定范围内的字符（如 `a` 到 `z` 的任意小写字母）         |

### 元字符

| 元字符 | 含义                              | 等效表达式       |
| ------ | --------------------------------- | ---------------- |
| `.`    | 匹配**单个**除换行符外的任意字符  | `[^\n]`          |
| `\d`   | 匹配**0-9**的任意数字             | `[0-9]`          |
| `\D`   | 匹配**非数字**的单个字符          | `[^0-9]`         |
| `\s`   | 匹配**空白字符**（空格/制表符等） | `[ \t\n\r\f\v]`  |
| `\S`   | 匹配**非空白**的单个字符          | `[^ \t\n\r\f\v]` |
| `\w`   | 匹配**字母/数字/下划线**          | `[a-zA-Z0-9_]`   |
| `\W`   | 匹配**非**字母/数字/下划线的字符  | `[^a-zA-Z0-9_]`  |

## 重复限定符（量词符）

重复限定符（量词符）用于指定其前面的字符和组合项连续出现多少次。

| 限定符  | 含义                 | 等效表达式 |
| ------- | -------------------- | ---------- |
| `?`     | 匹配0次或1次         | `{0,1}`    |
| `*`     | 匹配0次或多次        | `{0,}`     |
| `+`     | 匹配1次或多次        | `{1,}`     |
| `{n}`   | 匹配正好n次          | -          |
| `{n,}`  | 匹配至少n次          | -          |
| `{n,m}` | 匹配n到m次（含两端） | -          |

## 定位符

定位符用来指定要匹配的字符串出现的位置。

| 定位符 | 含义                             |
| ------ | -------------------------------- |
| `^`    | 匹配字符串的起始位置             |
| `$`    | 匹配字符串的结束位置             |
| `\b`   | 匹配单词边界（空格或字符串首尾） |
| `\B`   | 匹配非单词边界的位置             |

## 选择匹配符

正则表达式用符号 `|` 来表示 **或**，也叫做分支条件，当满足正则表达里的分支条件的任何一种条件时，都会当成匹配成功。

```python
import re

print(re.findall(r"土豆|马铃薯", "土豆和马铃薯是同一种东西"))  # ['土豆', '马铃薯']
```

## 分组组合

重复限定符（量词符）是作用在与它相邻的最左边的一个字符，正则表达式中可以使用小括号 `()` 来做分组，也就是括号中的内容会作为一个整体。

### 捕获分组

我们可以使用 `group()` 方法匹配对象方法，从一个分组中获取匹配的文本。正则表达式字符串中的第一对括号是第 1 组，第二对括号是第 2 组，依次类推。向 `group()` 匹配对象方法传入整数就可以取得匹配文本的不同部分。向 `group()` 传入 0 或者不传入参数，将返回这个匹配的文本。

如果想要一次就获取所有的分组，我们可以使用 `groups()` 方法。

| 语法格式            | 分组类型   | 说明                                                         |
| ------------------- | ---------- | ------------------------------------------------------------ |
| `(pattern)`         | 非命名捕获 | 1. 捕获匹配的子字符串<br>2. 分组编号规则：<br>   - `0`：整个模式匹配的文本<br>   - `1`开始：按左括号顺序自动编号 |
| `(?P<name>pattern)` | 命名捕获   | 1. 将匹配的子字符串绑定到指定名称<br>2. 命名规则：<br>   - 不能含标点符号<br>   - 不能以数字开头<br>3. 替代语法：`(?'name'pattern)` |

代码示例：

```python
import re

# 非命名捕获
m = re.search(r"(\d+)-(\w+)", "123-abc")
print(m.groups())  # 输出: ('123', 'abc')

# 命名捕获
m = re.search(r"(?P<id>\d+)-(?P<name>\w+)", "123-abc")
print(m.groupdict())  # 输出: {'id': '123', 'name': 'abc'}
```

### 非捕获组

| 语法格式      | 类型说明                     | 功能描述                                                     | 示例                                                         |
| ------------- | ---------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| `(?:pattern)` | 非捕获分组                   | 匹配`pattern`但不捕获分组内容<br>（不占用分组编号）          | `小樱(?:同学|同志)`<br>等价于`小樱同学|小樱同志`             |
| `(?=pattern)` | 正向肯定预查<br>（零宽断言） | 匹配后面跟随`pattern`的位置<br>（不消耗字符，不包含`pattern`在结果中） | `Harmony(?=2|3)`<br>匹配`Harmony2`中的`Harmony`<br>不匹配`Harmony1` |
| `(?!pattern)` | 正向否定预查<br>（零宽断言） | 匹配后面不跟随`pattern`的位置<br>（不消耗字符，不包含`pattern`在结果中） | `Harmony(?!2|3)`<br>匹配`Harmony1`中的`Harmony`<br>不匹配`Harmony2` |

代码示例：

```python
import re

# (?:) 示例
re.findall("小樱(?:同学|同志)", "小樱同学和小樱同志")  # 结果: ['小樱同学', '小樱同志']

# (?=) 示例
re.search("Harmony(?=2|3)", "Harmony2").group()  # 结果: 'Harmony'

# (?!) 示例
re.search("Harmony(?!2|3)", "Harmony1").group()  # 结果: 'Harmony'
```

## 贪婪模式 vs. 非贪婪模式

- **贪婪模式**（默认）：尽可能匹配更多字符
- **非贪婪模式**：在量词后加 ? 使其尽可能少匹配

代码示例：

```python
import re

content = "abc111111abc"

# 贪婪匹配（匹配最长连续3-5位数字）
greedy_result = re.findall(r"\d{3,5}", content)  # 优先匹配5位
print("贪婪匹配结果:", greedy_result)  # 输出: ['11111']

# 非贪婪匹配（匹配最短连续3位数字）
non_greedy_result = re.findall(r"\d{3,5}?", content)  # 满足3位即停止
print("非贪婪匹配结果:", non_greedy_result)  # 输出: ['111', '111']
```
