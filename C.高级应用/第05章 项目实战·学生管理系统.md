# 目录结构

```perl
student_management/
├── app/                      # 核心应用代码
│   ├── __init__.py           # 初始化应用
│   │
│   ├── api/                  # API 路由分组
│   │   ├── __init__.py
│   │   ├── v1/               # 按版本来管理接口
│   │   │   ├── __init__.py
│   │   │   ├── student.py    # 学生相关路由（CRUD）
│   │   │   ├── course.py     # 课程相关路由
│   │   │   └── ...
│   │   └── v2/
│   │       └── ...
│   │
│   ├── core/
│   │   ├── __init__.py
│   │   ├── auth              # 认证工具
│   │   │   ├── __init__.py
│   │   │   └── jwt.py        # JWT实现
│   │   ├── config.py         # 项目配置（数据库、密钥等）
│   │   ├── database.py       # 数据库连接与会话管理
│   │   └── exceptions.py     # 异常拦截
│   │
│   ├── dao/                  # DAO 层是数据访问层，专门封装数据库操作（增删改查）
│   │
│   ├── main.py               # FastAPI 主入口（创建 app 实例）
│   │
│   ├── middlewares/          # 全局中间件 
│   │   ├── __init__.py
│   │   └── ...
│   │
│   ├── models/               # 数据库模型（SQLAlchemy）
│   │   ├── __init__.py
│   │   ├── student.py        # 学生模型
│   │   └── course.py         # 课程模型
│   │
│   ├── schemas/              # Pydantic 数据验证模型
│   │   ├── __init__.py
│   │   ├── student.py        # 学生请求/响应模型
│   │   └── course.py         # 课程请求/响应模型
│   │
│   ├── services/             # 业务逻辑层
│   │   ├── __init__.py
│   │   ├── student.py        # 学生业务逻辑
│   │   └── course.py         # 课程业务逻辑
│   │
│   └── utils/                # 工具函数
│   │   ├── __init__.py
│   │   ├── logger.py         # 日志配置
│   │   └── crypto.py         # 加密/解密
│   │
├── .env                      # 环境变量
├── Dockerfile                # 容器化部署（可选）
├── README.md                 # 项目说明
├── requirements.txt          # 依赖列表
└── tests/                    # 测试代码
    ├── __init__.py
    ├── test_students.py      # 学生路由测试
    └── test_courses.py       # 课程路由测试
```

```shell
$ mkdir -p app/{api/{v1,v2},models,dao,core/auth,schemas,services,middlewares,utils} tests && touch app/__init__.py app/main.py app/api/__init__.py app/api/v1/__init__.py app/api/v1/users.py app/api/v1/posts.py app/api/v2/__init__.py app/models/__init__.py app/models/student.py app/models/course.py app/core/__init__.py app/core/auth/__init__.py app/core/auth/jwt.py app/core/exceptions.py app/core/config.py app/core/database.py app/schemas/__init__.py app/schemas/student.py app/schemas/course.py app/services/__init__.py app/services/student.py app/services/course.py app/middlewares/__init__.py app/middlewares/timing.py app/middlewares/logging.py app/utils/__init__.py app/utils/logger.py app/utils/crypto.py tests/__init__.py tests/test_students.py tests/test_courses.py .env requirements.txt README.md Dockerfile
```
